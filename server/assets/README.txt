PDF REPORT LOGO
===============

To show your real WCD logo on the downloaded PDF report, drop your logo image
into THIS folder and name it exactly:

    logo.png       (preferred)
    logo.jpg       (also accepted)

Requirements:
- Must be a PNG or JPG. SVG is NOT supported by the PDF engine.
- A square image (e.g. 256 x 256) looks best, since it sits in a circular badge.

That's it — no code changes. The next PDF you download will use it.
If no logo file is present here, the report falls back to a placeholder mark.
